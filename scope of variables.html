Scope of Variables

LOCAL VARIABLES

It can be accessed only within the function or block in which they are declared.Here scope of that variable is from starting to ending of function in which they declared.

GLOBAL VARIABLES

Global variables are those variables which are declared outside of all the functions or block and can be accessed globally in a program.

Global Variable: Outside of all the functions
Local Variable: Within a function block
